# pycaret_streamlit

## Wine Web App PyCaret and Streamlit Project

### First intall...

Install pycaret:
pip3 install pycaret[full]

Install Streamlit
pip3 install streamlit

### To run...
Go to terminal and navigate to where you 
downloaded this repo. 

Type: streamlit run extra_tree_model.py
